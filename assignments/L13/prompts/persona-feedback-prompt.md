# LLM Persona Feedback Prompt

다음 웹 애플리케이션은 AIOSS 수업 과제를 위한 Feature Flag 기반 A/B 테스트 대시보드이다.

## 실험 대상

- Experiment Name: homeHeroCopy
- Variant A: 기존 메인 문구
- Variant B: 실험 목적을 더 명확히 설명하는 개선 문구
- 주요 행동: 사용자가 대시보드에서 실험 variant를 확인하고 CTA 버튼을 클릭하는지 확인한다.

## 요청

서로 다른 사용자 패턴을 가진 10명의 가상 사용자를 생성하고, 각 사용자가 다음 기준으로 피드백을 작성하도록 해줘.

1. 사용자 역할
2. 사용 패턴
3. 수행 시나리오
4. 만족도 점수 1~5점
5. 과업 성공 여부
6. 정성 피드백
7. 개선 제안

## 조건

- 10명의 사용자는 모두 다른 페르소나를 가져야 한다.
- 초급 사용자, 모바일 사용자, QA, DevOps, 보안 리뷰어, PM, 접근성 관심 사용자 등을 포함한다.
- 피드백은 실제 실험 기록처럼 표 형태로 정리한다.
- A/B 테스트 결과를 Pivot 또는 Persevere 결정에 활용할 수 있도록 작성한다.
