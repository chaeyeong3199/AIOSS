# Getting Started

AIOSS에 오신 것을 환영합니다.

## 사전 준비

- Git
- Python 3.10+
- GitHub 계정

## 최초 설정

1. 저장소를 클론합니다.
2. 필요하다면 가상환경을 생성하고 활성화합니다.
3. 프로젝트 안내에 따라 의존성을 설치합니다.
4. 기본 스크립트를 실행해 결과가 정상인지 확인합니다.

## 처음 해보면 좋은 작업

- 프로젝트 개요와 과제 안내를 먼저 읽습니다.
- 지표 수집 스크립트를 1회 실행해 산출물을 확인합니다.
- 작은 문서 수정 또는 버그 수정 PR을 생성해봅니다.
