# 프로젝트 제안서 (Project Proposal) 

## 1. 기본 정보 (Basic Info)

- **프로젝트명:** Gaussian Pruning과 Importance Sampling을 적용한 GPS-Gaussian 경량화 및 가속화

- **개요:** GPS-Gaussian 모델의 불필요한 가우시안을 제거(Pruning)하고 핵심 영역에 샘플링을 집중하는 최적화 파이프라인 구현. 메모리 사용량 감소 및 렌더링 속도 향상이 목표이며, 분석부터 평가까지 단계별 파라미터 조정을 지원함.

- **동기:** 초실감 입체 공간 미디어 상용화 시 발생하는 대규모 연산 및 VRAM 병목 문제 해결. 모바일이나 VR/AR 등 제한된 환경에서도 실시간 고품질 뷰 합성을 가능하게 하여 산업적 적용성을 높임.

- **예상 결과물:** 원본 화질(PSNR, SSIM)을 유지하면서 메모리 사용량 50% 이상 감소 및 렌더링 속도 2배 이상 향상된 최적화 모델 코드. 벤치마크 결과 및 실험 데이터를 포함한 상세 성능 평가 보고서.

---

## 2. 주요 기능 (Key Features)

### 2.1 가우시안 Pruning

- 시각적 기여도가 낮은 가우시안을 자동 식별 및 제거: 렌더링 기여도 기반 중요도 점수(위치, 크기, 색상, 불투명도 등 고려)를 계산하여 임계값 이하의 가우시안 제거.

- 프루닝 비율 조정: 10%~90% 범위에서 프루닝 강도를 단계별로 조정하고 실시간 미리보기를 지원하여, 메모리 감소와 품질 간의 트레이드오프 조절 가능.

- 후처리 보정: 제거된 가우시안으로 인한 시각적 결함을 최소화하기 위해 주변 가우시안 속성 보정 알고리즘 적용.

### 2.2 Importance Sampling 기반 샘플링

- 고주파 디테일 영역에 샘플링 자원을 집중: 엣지 검출 및 텍스처 복잡도 분석을 통해 중요도 맵을 생성하고, 디테일이 풍부한 영역에 샘플 추가 할당.

- 동적 샘플링 전략: 프레임별/뷰별 중요도를 실시간 재평가하여 카메라 이동이나 씬 변화에 적응적으로 대응함.

- 샘플링 효율 최적화: 전체 샘플 수를 유지하면서 분배를 최적화하여 배경 연산은 줄이고 디테일 품질은 향상시킴.

### 2.3 통합 최적화 파이프라인

- 두 기법을 결합한 실시간 적용 및 성능 측정: 모듈화된 파이프라인을 통해 모델 로딩부터 최적화 적용, 성능 측정까지 자동화 지원.

- 피드백 루프: 최적화 결과를 실시간 모니터링하며, 품질 지표 하락 시 프루닝 강도를 낮추는 등 자동 튜닝 기능 포함.

- 사용자 인터페이스: CLI 및 스크립트 기반 제공(향후 GUI 확장 고려).

## 3. 기술 스택 (Tech Stack)

- **Frontend:** (해당 없음) 연구 및 개발 중심. 향후 필요 시 실시간 모니터링용 웹 대시보드 고려.

- **Backend:** Python, PyTorch. GPU 가속을 활용한 모델 최적화 구현. 필요시 OpenGL/Vulkan 통합.

- **Database:** 실험 결과, 파라미터, 성능 지표 기록용 SQLite (대규모 실험 시 PostgreSQL 확장).

- **Deployment:** 초기 로컬 GPU 환경 개발 후, 대규모 실험 시 클라우드 GPU(AWS, GCP 등) 활용. Docker를 통한 환경 일관성 유지.
